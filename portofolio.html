Portofolio Proyek

Website Pengelolaan Kembali Minyak Jelantah - Greasycle

Deskripsi Proyek

Greasycle adalah platform berbasis web yang dirancang untuk membantu masyarakat dalam mengelola minyak jelantah secara ramah lingkungan. Sistem ini menyediakan fitur penjadwalan penjemputan minyak bekas serta edukasi mengenai proses daur ulang menjadi biodiesel dan produk ramah lingkungan lainnya.

Fitur Utama

  • Registrasi & Login Pengguna atau Mitra
  • Form Penjadwalan Penjemputan Minyak Jelantah
  • Dashboard Admin untuk Monitoring Data
  • Halaman Edukasi Lingkungan
  • Laporan Statistik Pengumpulan Minyak

Teknologi yang Digunakan

HTML CSS JavaScript PHP MySQL

Tujuan Proyek

Proyek ini bertujuan untuk mengurangi pencemaran lingkungan akibat pembuangan minyak jelantah sembarangan serta mendukung konsep ekonomi sirkular melalui pemanfaatan limbah menjadi energi terbarukan.
